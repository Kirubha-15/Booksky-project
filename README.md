# Booksky-project
A JavaScript-based web app to add and delete books with details like title, author, and description, demonstrating DOM manipulation and dynamic UI updates.

🚀 Features
➕ Add new books with:
Title
Author

Description
❌ Delete books from the list
⚡ Instant UI updates without page refresh
📱 Simple and responsive design

🛠️ Tech Stack
HTML
CSS
JavaScript

📸 How It Works
Enter book details in the form
Click Add to save the book
Book appears instantly on the screen
Click Delete to remove a book

🎯 Learning Outcomes
DOM Manipulation
Event Handling in JavaScript
Dynamic Content Rendering
Basic Frontend Development

📌 Future Improvements
🔍 Search books

✏️ Edit book details
